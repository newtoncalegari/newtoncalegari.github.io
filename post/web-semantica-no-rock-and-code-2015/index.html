Web Semântica no Rock and Code 2015

December 1, 2015 ~ 2 min read

No último final de semana (28/11/2015) aconteceu o Rock and Code 2015 em Aracaju, SE - nos mesmos moldes do FrontInAracaju - organizado pelo Italo e pelo Julio. Que evento show!

Como na edição passada, também tivemos a banda e o Juarez Filho cuidando da apresentação dos palestrantes no estilo talk-show.

(Foto disponível na página do FrontInAracaju no facebook)

A programação era ótima, muitas palestrantes de qualidade, como o Maujor, Ciro Nunes, Gui Farias, Devid Marques, André Garzia (o cara que sabe tudo sobre scripts, sejam eles para filmes ou então em Javascript :-) ), Bernard de Luna, Giovanni Keppelen, Jaydson Gomes, entre outros. Infelizmente não consegui assistir todas as palestras, mas pelo que acompanhei, foram ótimas.

O evento estava muito bom. Concordo com o que o Juarez disse: “We had a blast!”

É isso que rola no mesmo palco das palestras entre um palestrante e outro no #RockAndCode. Muito foda! Quem ainda não foi, recomendo não perder o próximo! sai da frente @italowaxman kkkkk

A video posted by Guilherme Farias (@guiky) on Nov 30, 2015 at 10:45am PST

Foi uma oportunidade incrível para conhecer muita gente, conversar bastante e voltar para casa com ideias e projetos a serem desenvolvidos.

Na minha apresentação busquei mostrar os tópicos iniciais para que desenvolvedores web passem a publicar dados estruturados na Web, para formar a Web de Dados.

Mostrei alguns exemplos de uso dos vocabulários do Schema.org pelo Google, como o Google Rich Snippets, além de apresentar as especificações RDFa e JSON-LD(http://www.w3.org/TR/json-ld/). Coloquei abaixo os slides da palestra.

Web Semântica para desenvolvedores: RDFa, JSON-LD e schema.org from Newton Calegari

Mais uma vez, parabéns a todos os envolvidos!

Newton Calegari Líder de projetos e pesquisador trabalhando com assuntos relacionados a Open Web Platform e Web Semântica no W3C Brasil e Ceweb.br
