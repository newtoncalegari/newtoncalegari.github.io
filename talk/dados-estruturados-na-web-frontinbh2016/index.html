Palestra: A Semântica na Web e os dados estruturados no Google

August 15, 2016

Palestra realizada no FrontInBH 2016, onde apresentei sobre os fundamentos de Web Semântica e os primeiros passos para aplicar na rotina de desenvolvimento

Palestra sobre Web Semântica voltada para desenvolvedores realizada no FrontInBH 2016, na qual apresento sobre a evolução da Web (Web de Documentos para Web de Dados), as aplicações e usos práticos de semântica na Web.

Apresento também sobre RDFa, JSON-LD e Schema.org, mostrando exemplos de como o Google utiliza Web Semântica para melhorar seus produtos.
